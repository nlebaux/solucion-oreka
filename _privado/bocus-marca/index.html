Bocus · sistema de marca

Bocus es una marca gastronomica inspirada en el espiritu Bauhaus: forma y funcion trabajando juntas. El objetivo de este manual es fijar una identidad que se vea precisa, racional y moderna, pero tambien calida, tangible y profundamente culinaria.

ADN: Bauhaus reinterpretado Estado: v2 privada Ultima actualizacion: 2026-05-31
Proposito Fusionar arte, diseño y gastronomia para crear experiencias funcionales, esteticas y honestas.
Promesa Cada plato debe sentirse como una composicion equilibrada entre sabor, forma y concepto.
Personalidad Minimalista, moderna, precisa, intelectual, experimental.

ADN

Lo que no se negocia

  • Funcion primero. Nada es decorativo si no ayuda a ordenar, leer, elegir o entender.
  • Geometria con proposito. Circulos, modulos y superposiciones deben reforzar estructura, no ruido.
  • Calidez material. Aunque la marca es racional, no debe sentirse fria ni clinica.
  • Honestidad visual. El producto y el sistema hablan claro. No hay maquillaje innecesario.

Voz

Como habla Bocus

  • Directo. Explica, no declama.
  • Racional. La marca describe valor, proceso y criterio antes que sensaciones vacias.
  • Seguro. No ruega atencion ni sobreactua premium.
  • Contenido limpio. Menos slogans, mas precision.

Tipografia

Sistema tipografico

  • Titulares. Futura como referencia conceptual; alternativas viables: ITC Avant Garde Gothic, Poppins o Jost cuando haga falta un sistema web mas flexible.
  • Texto corrido. Montserrat o Helvetica Neue para lectura continua y materiales digitales.
  • Jerarquia. Mayusculas controladas, tracking amplio en titulos y subtitulos sobrios.
  • Regla. La tipografia debe verse limpia, modular y decidida. Nunca timida.

Composicion

Como se arma una pieza

  • Reticula visible. La composicion debe sentirse construida, no improvisada.
  • Formas superpuestas. Cintas, circulos y bloques deben aportar ritmo y foco.
  • Producto protagonista. La fotografia entra integrada, nunca como parche.
  • Espacio negativo. Parte de la sofisticacion viene de no saturar.

Do / Don't

Criterio rapido de control

  • Si. Modulos claros, contraste limpio, acentos medidos, fondos terrosos, producto bien recortado.
  • Si. Etiquetas y mensajes cortos, utiles y seguros.
  • Si. Packaging y piezas que se sientan industriales pero nobles.
  • No. Sombras suaves de moda, gradientes dulzones o exceso de efectos.
  • No. Tipografias decorativas o storytelling blando.
  • No. Fotografia sin estructura o composiciones que pierdan el rigor Bauhaus.
