# Solucion Oreka

Pagina web inicial para `solucionoreka.com`, lista para publicar en Netlify.

## Como publicarla

1. Sube estos archivos a un repositorio de GitHub.
2. Entra a Netlify y elige **Add new site**.
3. Selecciona **Import an existing project**.
4. Conecta tu cuenta de GitHub y elige este repositorio.
5. En la configuracion de publicacion, deja:
   - Build command: vacio
   - Publish directory: `.`
6. Publica el sitio.

## Conectar solucionoreka.com

1. En Netlify, entra al sitio publicado.
2. Abre **Domain management**.
3. Elige **Add a domain** y escribe `solucionoreka.com`.
4. Netlify te mostrara los registros DNS que debes poner donde compraste el dominio.
5. Copia esos registros exactamente en el panel del proveedor de tu dominio.
6. Espera la activacion. Puede ser rapido, pero a veces tarda varias horas.

## Cambios faciles

- El texto principal esta en `index.html`.
- Los colores y estilos estan en `styles.css`.
- El logo usado por la pagina esta en `assets/logo-solucion-oreka.png`.
- El correo actual es `hola@solucionoreka.com`; puedes cambiarlo por el correo real.
